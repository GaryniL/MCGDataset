Definition of a Unit Motion

Data Structure

  • File Format：CSV, each CSV file include all groups.
  • File Counts：2 (Count table & Time table)
  • Groups：36 (EL1-EL2: 18, EL2-EL2: 18)
  • Participants per Group：2, total 72 participants.
  • Media Condition：Audio, HandVis, Face-to-Face
  • Data Type：3
    • Counts of total unit motion in each session (Threshold: 0.0, 0.05~0.5) [Table 1]
      • Unit Motion Count：count of unit motion of each participants in each session.
      • Threshold: 0.0, 0.05, 0.1, 0.15, 0.2, 0.25, 0.3, 0.35, 0.4, 0.45, 0.5
    • Total & Average time of unit motions in each session (Threshold: 0.0, 0.05~0.5) [Table 2]
      • UnitMotion_Time: Total time of unit motions in each session participants made.
      • Average time of unit motions in each session participants made.
      • Threshold: 0.0, 0.05, 0.1, 0.15, 0.2, 0.25, 0.3, 0.35, 0.4, 0.45, 0.5

Definition of Unit Motion

Definition of a unit motion: in the beginning, the speed of the motion should be slow (i.e., starting from speed zero or below the threshold we defined) and then the speed increases because of user using their gesture, after the motion speed reaching to crest, it would decrease again to zero or below threshold.

Reference：Wang, H. C., & Lai, C. T. (2014, April). Kinect-taped communication: using motion sensing to study gesture use and similarity in face-to-face and computer-mediated brainstorming. In Proceedings of the 32nd annual ACM conference on Human factors in computing systems (pp. 3205-3214). ACM.
