Idea Count

Each pair was asked to discuss three brainstorming tasks of similar format and difficulty in three different conditions. The tasks are: How do you think if all humans “having an extra thumb”, “having an extra eye” and “having wings” in the year 2020. Participants don’t know the brainstorming task before each brainstorming session starts. When the session start, we will give participants a paper with the task description, and they have 5 minutes to read and understand the task.

Definition of idea is the semantic topic related to the brainstorming topic we assigned to participants. Coder will only count idea once, idea will refer to the speaker who first blurt it.

Data Structure

  • File Format：CSV, each CSV file include all groups.
  • File Counts：1
  • Groups：36 (EL1-EL2:18,EL2:EL2:18)

  • Media Condition：Audio, HandVis, Face-to-Face
