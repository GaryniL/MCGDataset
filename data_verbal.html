Words & Sentneces

A word means a single distinct meaningful element of speech, and sentence is a set of words that is complete in itself. For each utterance, we used an auto-separation python program to separate utterances into sentences by period (.), question mark (?) and exclamation mark (!). By grammar rules, these symbols appear at the end of a sentence. Moreover, we calculated the number of words by splitting those sentences with space and comma.


Data Structure

  • File Format: CSV, each CSV file includes all groups.
  • File Counts: 1
  • Groups: 36 (EL1-EL2:18,EL2:EL2:18)
  • Participants per Group: 2, total 72 participants.
  • Media Condition: Audio, HandVis, Face-to-Face
  • Data Type: Counts of words & sentences (Threshold:0.0,0.05~0.5) [Table 1]
